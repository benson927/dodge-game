CORE GAMEPLAY // 核心玩法

  • 直覺的操作體驗：捨棄繁雜的輸入介面，遊戲全面採用純 WASD 鍵盤控制。我們將飛船機動性調校至滑順與精準之間的完美平衡。
  • 無止盡的生存挑戰：挑戰者需躲避不斷從上方落下的隕石風暴。活得越久，難度曲線急遽提升（隕石速度與生成頻率遞增），創造容易上手卻難以精通的「再玩一局」成癮感。
  • 動態計分系統：以「存活時間」來量化您的實力，自動記錄突破極限的 최고分數 (High Score)。

AESTHETICS // 次世代視覺設計

  • Glassmorphism (玻璃擬物化 UI)：結合半透明磨砂玻璃與邊緣反光，打造出現代而高質感的科幻駕駛艙面板。
  • 多層次視差星空 (Parallax Background)：背景採用 3 層不同速度與大小的星空圖層連動，當飛船移動與推進時，帶來極具深度的宇宙沉浸感。
  • 豐富的光效粒子 (Particle System)：從飛船推進的「藍色尾焰」、隕石劃過的「燃燒軌跡」，到碰撞時的「高能爆破火花」，提供絕佳的重擊回饋 (Juiciness)。
  • 濾鏡與字體工程：字體採用 "Orbitron" + "Chakra Petch"，搭配 Glitch 故障動畫特效，在 Game Over 時視覺衝擊力十足。

POWER-UPS // 戰術增幅道具

在極限圖存中，隨機出現的道具將成為你逆轉局勢的關鍵。我們設計了三款改變戰局的核心能力：

⛨

SHIELD

啟動能量光環，免除一次致命碰撞。破盾時會引發保護性衝擊波震動。

⌚

TIME WARP

全畫面進入紫色扭曲維度，降低隕石與背景流速長達 10 秒，換取喘息空間。

⚠

CLEAR BOMB

釋放高能脈衝引爆畫面內所有隕石，並同時獲得額外 50 秒的獎勵分數。

MARKET POTENTIAL // 商業化與未來展望

  • 跨平台無縫移植：HTML5/Canvas 架構可以輕易封裝成 WebView 放到 iOS/Android 手機雙平台上。結合手指觸控拖曳 (已實作基礎) 可成為殺時間神作。
  • 社群與排行榜擴充：將 High Score 接上伺服器，推出「每週全球百大排名榜」，增加玩家炫耀與好勝的動機。
  • 微交易與客製化 (Monetization)：利用遊戲內獲得的代幣或 IAP (應用內購買)解鎖「飛船造型 (Skins)」、「隕石樣式」及「特殊的尾焰光影特效」。
